Detailed Description

template<typename... Args>
class BalisongEngine::BalisongEngineFramework::ActionEvent< Args >

An event object may call multiple slots with the same signature.

You can connect functions to the event which will be called when the Invoke() method on the event object is invoked. Any argument passed to Invoke() will be passed to the given functions.
